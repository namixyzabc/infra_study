# Vimの基本

**終了コマンド:**
- `:q` - 終了
- `:q!` - 保存せず強制終了
- `:w` - 保存
- `:wq` または `:x` または `ZZ` - 保存して終了

## モードの概念
- **ノーマルモード** - 起動時のデフォルト（移動・操作）
- **インサートモード** - 文字入力（`i`, `a`, `o`で入入）
- **ビジュアルモード** - 範囲選択（`v`で入入）
- **コマンドモード** - コマンド実行（`:`で入入）

**モード切替:**
- `Esc` - ノーマルモードに戻る

## カーソル移動（ノーマルモード）
```
h j k l           # ←↓↑→
w / b             # 次の単語 / 前の単語
0 / $             # 行頭 / 行末
gg / G            # ファイル先頭 / 末尾
:数字 または 数字G  # 指定行へジャンプ
Ctrl+f / Ctrl+b   # 1画面前 / 後
```

## 編集開始（インサートモード移行）
```
i    # カーソル位置から挿入
a    # カーソルの次から挿入
o    # 下に新しい行を作成
I    # 行頭から挿入
A    # 行末から挿入
O    # 上に新しい行を作成
```

## 削除・コピー・貼り付け
```
x / X       # 1文字削除（後/前）
dd          # 1行削除
dw          # 単語削除
D           # カーソルから行末まで削除
yy          # 1行コピー
yw          # 単語コピー
p / P       # 貼り付け（後/前）
u           # 元に戻す（Undo）
Ctrl+r      # やり直し（Redo）
```

## 検索・置換
```
/文字列          # 前方検索
?文字列          # 後方検索
n / N           # 次/前の検索結果
:%s/前/後/g     # 全置換
:%s/前/後/gc    # 確認しながら全置換
:noh            # 検索ハイライト解除
```

## ビジュアルモード
```
v               # 文字単位選択
V               # 行単位選択
Ctrl+v          # 矩形選択
```
選択後: `d`削除, `y`コピー, `>`インデント

## 便利なコマンド
```
.               # 直前の操作を繰り返し
r               # 1文字置換
cc              # 行を置き換え
>>  / <<        # インデント / アンインデント
:set number     # 行番号表示
:set paste      # ペーストモード
:e ファイル名    # ファイルを開く
:sp / :vs       # 水平/垂直分割
```

## 数字との組み合わせ
```
3dd             # 3行削除
5yy             # 5行コピー
10j             # 10行下へ移動
```

## 最低限覚えるべきこと
1. `i` で入力、`Esc` で抜ける
2. `:wq` で保存終了、`:q!` で破棄終了
3. `h j k l` で移動
4. `dd` で行削除、`yy` でコピー、`p` で貼り付け
5. `u` で元に戻す
