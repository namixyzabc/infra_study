## **SPA以前 vs SPA の主な違い**

| 項目 | SPA以前 | SPA |
| ----- | ----- | ----- |
| ページ遷移 | 全画面リロード | 画面遷移なし |
| 速度 | 遅い（毎回読み込み） | 高速（初回のみ重い） |
| ユーザー体験 | 途切れる | 滑らか |
| 開発 | シンプル | 複雑 |
| SEO | 良好 | 困難 |
| JavaScript依存 | 不要〜部分的 | 完全依存 |

## **代表例**

* SPA以前: 従来のWebサイト、jQuery \+ AJAX  
* SPA: React、Vue、Angularアプリ

## **一言で言うと**

SPA以前: ページをめくる本 📖  
SPA: スマホアプリのような体験 📱
